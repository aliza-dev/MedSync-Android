## 📖 Overview

**MedSync** is a role-based healthcare Android application that connects **Patients** and **Doctors** through a unified platform — enabling appointment booking, treatment tracking, and digital billing, all powered by Firebase Authentication and Realtime Database.

Submitted as an academic project for **Mobile App Development** at the **University of Agriculture, Faisalabad**, MedSync showcases practical mobile development with secure dual-role access, Material Design UI, and real-time cloud data synchronization.

## ✨ Core Features

### 👤 For Patients

- 🔐 **Secure Registration & Login** — Email/password auth with role-based access
- 👤 **Profile Management** — View name, email, and age from your dashboard
- 📅 **Appointment Booking** — Schedule visits with date/time picker and problem details
- 💊 **Treatment History** — Review diagnosis, prescriptions, and progress notes
- 🧾 **Bills History** — Access medical bills from completed appointments

### 🩺 For Doctors

- 🏠 **Doctor Dashboard** — Profile overview with quick navigation to all tools
- ⏳ **Pending Appointments** — Review and manage incoming patient requests
- ✅ **Accept / Reject** — Approve or decline today's scheduled appointments
- 📝 **Clinical Updates** — Add diagnosis, prescription, and treatment progress
- 💰 **Medical Billing** — Generate itemized bills (consultation + medicine fees)
- 📚 **Patient History** — Browse completed records across all patients
